# PATCH API DOCS
**1. Update Profile**
* **URL**
    `/api/update`

* **METHOD**
    `PATCH`

* **PASSED OBJETCS** <br />

    required atleast one field to update

    `firstName,` <br />
    `lastName,` <br />
    ` password,` <br />
    `email,`

* **Success Response**<br />
    * **Code:** `201` <br />
      **Message:** `Profile updated` <br />
      **data:** `{ id: "user_id", firstName, lastName, email }`

* **Error Response**
    * **Code:** `401` <br />
      **Message:** `Authorization Required` <br />

* **Required Filed**
     * **Auth Need : True**
    * **Admin Auth Need : False**
